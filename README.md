# 🏏 IPL Data Analysis Project (2008-2024)

## Objective
To analyze IPL match data and find key insights about 
team performance, player stats and match outcomes.

## Tools Used
Python, Pandas, Matplotlib, Seaborn, Jupyter Notebook

## Dataset
IPL Complete Dataset from Kaggle
- matches.csv — 1095 matches
- deliveries.csv — 260920 deliveries

## Key Findings
1. Mumbai Indians have the most wins in IPL history with 144 wins
2. Teams prefer to field after winning toss
3. Toss winners won 50.59% of matches
4. Virat Kohli is the highest run scorer in IPL history
5. Chris Gayle is the biggest six hitter in IPL history

## Analyses Done
1. Top 10 teams by total wins
2. Toss impact on match result
3. Top 10 venues by matches hosted
4. Top 10 batsmen by total runs
5. Top 10 bowlers by total wickets
6. Season wise wins for Mumbai Indians
7. Player of the match award winners
8. Top 10 six hitters

## How to Run
1. Download both CSV files and the notebook
2. Open in Jupyter Notebook or Google Colab
3. Run all cells in order
